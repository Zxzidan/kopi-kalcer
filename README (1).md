# ☕ Kopi Kalcer

![Logo Kopi Kalcer](assets/img/preview)

## 📌 Tentang Project

**Kopi Kalcer** adalah sebuah website UMKM yang menampilkan produk kopi lokal dengan konsep modern dan kekinian. Website ini dibuat untuk membantu mempromosikan usaha kopi yang didirikan pada tahun **2026** di **Surabaya**.

Project ini bertujuan untuk:
- Menampilkan produk kopi secara menarik
- Memudahkan pelanggan melihat daftar menu
- Mengelola data produk melalui form input
- Menyajikan data dalam bentuk tabel

---

## 🚀 Fitur Utama

- 📝 **Form Input Data Produk**
  - Menambahkan produk kopi baru
  - Input nama, harga, dan deskripsi

- 📊 **Tampilan Data (Table)**
  - Menampilkan daftar produk
  - Data tersusun rapi dan mudah dibaca

- 🎨 **UI Responsif**
  - Menggunakan Bootstrap
  - Tampilan menyesuaikan perangkat (desktop & mobile)

- 🖼️ **Branding UMKM**
  - Menggunakan logo Kopi Kalcer
  - Desain modern dan sederhana

---
